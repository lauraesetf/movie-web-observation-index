# movie-web-observation-index
该仓库主要用于收集和整理影视相关网站的公开观察索引信息，覆盖电影、电视剧、综艺及动漫等多类型站点。整理内容仅限站点层级的客观资料，不包含任何影视内容文件、播放链接或资源引用。所有资料均整理自公开渠道，目的在于形成可长期维护的站点观察索引，便于后续对照、分析与查询。本项目不参与站点运营，也不对站点内容质量、版权属性或更新频率作出判断，仅作为中性的信息整理与观察归档用途存在。
